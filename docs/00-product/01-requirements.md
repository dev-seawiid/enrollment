## Description

- 서비스 요구사항에 대한 **정확한 이해**
- 기능의 **효율적이고 적시적인 구현**
- **버그를 최소화**하는 높은 코드 품질
- 유연한 변경 사항 대응을 위한 **재사용 및 확장 가능한** 애플리케이션 구조 설계
- 유저가 사용하기 편한 **사용성**
- 협업을 위한 **코드 및 문서 가독성**

이를 위해 본 과제에서는 강의 등록 및 신청에 필요한 UI를 개발하려고 합니다.

- 회원 가입
- 강의 등록
- 강의 신청

---

## Feature Specification

### 1. 회원 가입

고객이 웹에서 회원 정보를 입력하고 가입하는 기능입니다.

- 회원 가입 화면에서 필요한 정보를 입력하고 **가입하기** 버튼을 클릭하면 가입이 완료됩니다.
- 회원 가입이 완료되면 강의 목록으로 이동합니다.
- 가입 시 입력하는 회원정보는 아래와 같습니다.

| 항목        | 설명                                                                                                              |
| ----------- | ----------------------------------------------------------------------------------------------------------------- |
| 이름        | 회원 이름                                                                                                         |
| 이메일      | 회원 이메일                                                                                                       |
| 휴대폰 번호 | 회원 휴대폰 번호                                                                                                  |
| 비밀번호    | 로그인을 위한 비밀번호<br>• 최소 6자 이상 10자 이하<br>• 영문 소문자, 대문자, 숫자 중 최소 두 가지 이상 조합 필요 |
| 회원 유형   | 수강생 / 강사                                                                                                     |

**화면 예시**

```
┌─────────────────────────┐
│         회원 가입         │
│                         │
│  이름                    │
│  ┌─────────────────┐    │
│  │ 홍길동           │    │
│  └─────────────────┘    │
│                         │
│  이메일                  │
│  ┌─────────────────┐    │
│  │ hong@weolbu.com │    │
│  └─────────────────┘    │
│                         │
│  휴대폰 번호              │
│  ┌─────────────────┐    │
│  │ 010-1234-5678   │    │
│  └─────────────────┘    │
│                         │
│  비밀번호                │
│  ┌─────────────────┐    │
│  │ ******          │    │
│  └─────────────────┘    │
│                         │
│  회원 유형               │
│  ◉ 수강생    ◉ 강사      │
└─────────────────────────┘
```

### 2. 강의 개설

강사로 가입한 회원이 자신의 강좌를 등록하는 기능입니다. 등록된 강좌는 수강생 회원이 신청할 수 있습니다.

- 강의 목록 화면에서 **강의 개설** 버튼을 클릭하면 본 화면으로 진입합니다. 강의 개설이 완료되면 다시 강의 목록 화면으로 이동합니다.
- 웹의 강의 등록 화면에서 강의 정보를 입력하고, **등록하기** 버튼을 클릭하면 등록이 완료됩니다.
- 수강생 회원은 강의를 등록할 수 없습니다.
- 강의 등록 시 입력하는 정보는 아래와 같습니다.

| 항목           | 설명                         |
| -------------- | ---------------------------- |
| 강의명         | 예) 너나위의 내집마련 기초반 |
| 최대 수강 인원 | 예) 10명                     |
| 가격           | 예) 200,000원                |

**화면 예시**

```
┌─────────────────────────┐
│         강의 등록         │
│                         │
│  강의명                  │
│  ┌─────────────────┐    │
│  │너나위의 내집마련  │    │
│  │기초반            │    │
│  └─────────────────┘    │
│                         │
│  수강 인원               │
│  ┌─────────────────┐    │
│  │ 10               │    │
│  └─────────────────┘    │
│                         │
│  가격                    │
│  ┌─────────────────┐    │
│  │ 200,000          │    │
│  └─────────────────┘    │
│                         │
│  ┌─────────────────┐    │
│  │     등록하기      │    │
│  └─────────────────┘    │
└─────────────────────────┘
```

### 3. 수강 신청

회원이 강의 목록을 확인하고 신청하는 기능입니다. 웹의 강의 목록에서 수강하고 싶은 강의를 선택하여 신청합니다.

- 강의 목록은 **무한 스크롤**이 가능합니다. 스크롤 다운 시 자연스럽게 하위 아이템이 노출되어야 합니다.
- **최근 등록순**, **신청자 많은 순**, **신청률 높은 순**으로 정렬이 가능합니다.
  - 신청자 많은 순: 현재 기준 신청자가 많은 강의 순으로 표시
  - 신청률 높은 순: 신청률 = 신청자 수 / 최대 수강 인원 수, 비율이 높은 강의부터 표시
- 강의 목록의 각 강의 아이템은 아래 정보를 표시합니다.

| 항목             | 설명                         |
| ---------------- | ---------------------------- |
| 강의명           | 예) 너나위의 내집마련 기초반 |
| 가격             | 예) 200,000원                |
| 강사명           | 예) 너나위                   |
| 현재 신청자 수   | 예) 8명                      |
| 최대 수강 인원수 | 예) 10명                     |

- 강의는 **여러 개를 선택하여 한 번에 신청**할 수 있습니다.
- 강사 회원도 수강 신청이 가능합니다.

**화면 예시**

```
┌──────────────────────────────┐
│           강의 목록            │
│                              │
│ ◉ 최근 등록순 ◉ 신청자 많은순  │
│              ◉ 신청률 높은순   │
│                              │
│ ☑ ┌────────────────────────┐ │
│   │ 너나위의 내집마련 기초반  │ │

│ ☑ ┌────────────────────────┐ │
│   │ 신도시투자 기초반         │ │
│   │ 강사명: 자음과모음 수강인원 5/10│ │
│   │                300,000원│ │
│   └────────────────────────┘ │
│                              │
│ ☑ ┌────────────────────────┐ │
│   │ 열반스쿨 기초반           │ │
│   │ 강사명: 너바나  수강인원 8/10│ │

│                              │
│   ┌────────────────────────┐ │
```

## Requirement

- **모바일 웹 기준**으로 구현합니다. UI는 와이어프레임을 바탕으로 자유롭게 구성하시면 됩니다.

### 기본 조건

- **Next App Router**를 사용하여 과제를 진행
- **Typescript**만 사용 가능
- 검증된 패키지는 자유롭게 사용 가능
- SSR(Server Side Rendering) 구현
- Semantic HTML을 준수하여 구조화된 마크업 작성
- Flexbox 또는 Grid Layout을 활용한 레이아웃 구성
- 로직 및 컴포넌트에 대한 테스트 코드 작성
- `README.md`에 프로젝트의 간단한 설명 작성
